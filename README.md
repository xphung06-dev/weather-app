# Weather App

A simple and responsive weather web application built using HTML, CSS, and JavaScript.  
This app uses the WeatherAPI to fetch real-time weather data for any location.

---

## Features

- Search weather by city or country
- Shows real-time temperature in Celsius
- Displays local time and date
- Weather condition with icon
- Dynamic background based on time (day, night, sunrise, sunset)
- Fully responsive design

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- WeatherAPI

---

## API Used

- https://www.weatherapi.com/
